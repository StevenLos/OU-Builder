# Project Input

Place the source OUTree CSV here.

This template folder includes `outree-layout-template.csv` as a starting point. Replace it with a real source CSV or rename it after editing.

Source CSVs must include `Depth0`. `Depth1` through `Depth9` are supported OU nesting levels and are optional when the source file does not need them.

Recommended naming:

```text
OUTree_<domain-with-dashes>-v<number>.csv
```

Example:

```text
OUTree_ad-example-com-v1.csv
```
