# Templates

This folder contains reusable templates for starting new OUBuilder project work.

Available folders:

- `my-project-template/` - project folder template with `input` and `output` folders.
- `my-project-template/input/outree-layout-template.csv` - starter source CSV layout with `Depth0` through `Depth9`.

Recommended project-template copy command from the package root:

```powershell
Copy-Item .\02-templates\my-project-template .\03-projects\my-ou-project -Recurse
```

The converter expects an OUTree source CSV. Normal output should be generated by `Convert-OUTreeToOUCsv.ps1`; do not hand-build the OU creation CSV unless you are troubleshooting.
